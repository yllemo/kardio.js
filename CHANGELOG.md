# Changelog

Alla viktiga förändringar i detta projekt dokumenteras i denna fil.

Formatet baseras på [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
och projektet följer [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-08

### Tillagt
- 🎯 Grundläggande kortdiagram-funktionalitet
- 🎨 Drag-and-drop interaktivitet för kort
- 🔗 Flexibla relationer med pilar och etiketter
- 📱 Responsiv design som fungerar på alla skärmstorlekar
- ⚡ Auto-initialisering från HTML-attribut
- 🎛️ Fyra layoutlägen: Fritt, Hierarki, Mindmap, Nätverk, Tidslinje
- 💾 Export/Import av JSON och DSL-format
- 🌈 10 inbyggda färgteman
- 🖱️ Interaktiv redigering av kort och relationer
- 📝 Enkel DSL-syntax för att definiera diagram
- 🔧 JavaScript API för programmatisk användning
- 📚 Omfattande dokumentation på svenska

### Tekniska funktioner
- Inga externa beroenden
- En enda JavaScript-fil (kardio.js)
- Automatisk laddning av Google Fonts (Syne + Space Mono)
- Automatisk CSS-injektion
- Cross-browser kompatibilitet
- CSP-kompatibel (inget `eval()` eller inline scripts)

---

**Format för framtida versioner:**

### Added (Tillagt)
### Changed (Ändrat)  
### Deprecated (Utfasad funktionalitet)
### Removed (Borttaget)
### Fixed (Fixat)
### Security (Säkerhet)
